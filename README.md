# RulesForge

AI coding rules generator for Claude Code, Cursor, Windsurf, Copilot.

[![Live Demo](https://img.shields.io/badge/Live_Demo-rulesforge.vercel.app-blue?style=for-the-badge)](https://rulesforge.vercel.app)

[Live Demo](https://rulesforge.vercel.app) | [Report Bug](https://github.com/keyflowcoreg/rulesforge/issues)

## Features

- Generate CLAUDE.md, .cursorrules, and AGENTS.md files instantly
- Choose framework, coding style, and team size to customize rules
- Free tier with copy-paste rules, Pro Pack with complete bundles
- Supports Next.js, React, Python, Go, and more
- x402 crypto payments (USDC on Base)

## Tech Stack

Next.js 16 · React 19 · TypeScript · Tailwind CSS v4 · Framer Motion · x402 Payments

## Getting Started

```bash
pnpm install
pnpm dev
```

Open [http://localhost:4008](http://localhost:4008) to view it locally.
